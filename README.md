## 目录结构

```
source/
├── preface/                     # 前言
├── chapter_01_intro/            # 公司制度概述
├── chapter_02_governance/       # 公司治理架构
├── chapter_03_charter/          # 公司章程与股东协议
├── chapter_04_equity/           # 股权结构设计
├── chapter_05_financing/        # 融资与资本运作
├── chapter_06_esop/             # 员工激励与ESOP
├── chapter_07_capital_market/   # 中国资本市场体系
├── chapter_08_compliance/       # 合规与风险管理
├── chapter_09_cases/            # 公司治理实践案例
├── chapter_10_trends/           # 未来趋势与制度创新
└── appendix/                    # 附录
```

## 本地构建

```bash
pip install -r requirements.txt
make html       # 构建 HTML
make serve      # 构建并在 localhost:8000 启动预览
```
