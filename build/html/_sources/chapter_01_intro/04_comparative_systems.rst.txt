===========================
不同法系下的公司制度比较
===========================

.. _chapter-01-comparison:

====================
全球公司法的主要法系
====================

全球公司法体系大致可分为两大法系：

**大陆法系（Civil Law）**
   以成文法为核心，法律规则主要由立法机关以法典形式制定。法官的角色是
   适用法律而非创造法律。主要代表：中国、德国、法国、日本、韩国。

**英美法系（Common Law）**
   以判例法为核心，法院的判决对后续案件具有约束力，成文法是对判例法的
   补充和系统化。主要代表：美国、英国、加拿大、澳大利亚、中国香港。

两大法系在公司制度上的差异，不仅体现在法律渊源上，更在治理结构、股东
权利、融资制度等具体规则层面形成了不同的路径。

==============================
大陆法系 vs 英美法系：核心差异
==============================

.. list-table:: 大陆法系与英美法系公司制度的核心差异
   :widths: 18 28 28 26
   :header-rows: 1

   * - 比较维度
     - 大陆法系（中国/德国/日本）
     - 英美法系（美国/英国）
     - 差异分析
   * - 治理结构
     - 双层制：董事会+监事会分离，
       监事会独立监督
     - 单层制：单一董事会，
       下设各专门委员会
     - 大陆法系强调独立监督机构；
       英美法系依赖独立董事
   * - 股权结构
     - 普遍存在控股股东，
       股权集中度高
     - 股权分散（美国），
       机构投资者主导
     - 大陆法系代理问题主要在
       大股东vs小股东
   * - 资本市场角色
     - 银行主导融资
       （主银行制度）
     - 资本市场主导，
       直接融资发达
     - 英美法系公司更依赖
       公开市场融资
   * - 职工参与
     - 德国强制的劳资共同
       决定制度（监事会含职工代表）
     - 一般不强制
       （美国无法律规定）
     - 德国模式在职工权益
       保障方面最为激进
   * - 法律渊源
     - 成文法典为主，
       判例仅作参考
     - 判例法为主，
       成文法为补充
     - 影响法律适用的
       可预测性和灵活性
   * - 公司设立
     - 注册制为主，
       但有实缴或认缴要求
     - 注册制，
       各州竞争（美国特拉华州）
     - 特拉华州因高效的公司
       法院而成为美国首选

====================
中国公司法的特色制度
====================

中国公司法虽然属于大陆法系，但在吸收两大法系经验的基础上形成了若干
具有中国特色的制度安排：

~~~~~~~~~~
监事会制度
~~~~~~~~~~

中国采取了类似德国的**双层制治理结构**，监事会作为独立的监督机构与
董事会平行运作。不同于德国的是，中国监事会中没有银行代表和劳方代表
的强制席位安排。2024年新公司法引入审计委员会替代方案，标志着中国在
治理结构上开始向灵活化方向发展。

~~~~~~~~~~~~~~
法定代表人制度
~~~~~~~~~~~~~~

法定代表人制度是中国公司法的独有特色。根据中国法律，每个公司必须设立
一名法定代表人，其在法律上代表公司对外从事民事活动。法定代表人的行为
后果直接由公司承担。

- 2024年新公司法将法定代表人范围从"董事长"扩大到"代表公司执行事务的
  董事或经理"
- 法定代表人的越权行为，公司仍需对善意第三人承担责任
- 这一制度在大陆法系其他国家（如德国、日本）并无对应设计

~~~~~~~~~~~~~~
注册资本认缴制
~~~~~~~~~~~~~~

2013年中国率先实行完全的注册资本认缴制，取消了最低注册资本限制和首期
出资比例要求。2024年新公司法进行了回调——规定股东出资期限不得超过五年，
体现了"宽进严管"的政策调整。

.. list-table:: 各国公司注册资本制度对比
   :widths: 20 20 20 20 20
   :header-rows: 1

   * - 国家
     - 最低注册资本
     - 出资期限
     - 出资形式
     - 资本维持要求
   * - 中国（2024后）
     - 无最低限额
     - 5年内缴足
     - 货币/实物/知识产权/
       股权/债权等
     - 严格，禁止抽逃出资
   * - 德国
     - 有限责任：2.5万欧元
       （实缴50%）
     - 无强制期限
     - 货币/实物需验资
     - 严格，有资本维持义务
   * - 日本
     - 无最低限额
     - 无强制期限
     - 货币/实物
     - 较宽松
   * - 美国（特拉华州）
     - 无最低限额
     - 无强制期限
     - 任何对价（含劳务）
     - 较宽松

================
开曼公司法的特点
================

开曼群岛（Cayman Islands）是全球最重要的离岸金融中心之一，其公司法
以英国普通法为基础，具有以下特点，使其成为中国企业境外上市的首选：

豁免公司（Exempted Company）
   开曼公司法将公司分为"本地公司"和"豁免公司"两大类。在开曼注册的
   境外上市主体通常采用**豁免公司**形式，其核心优势包括：
   - 可向开曼政府申请**免除未来20年税收**的证书（Tax Exemption）
   - 无需在开曼设立实际办公场所（可仅设注册代理人）
   - 股东名册不对外公开，保护隐私

灵活的治理结构
   - 允许采用单层制（仅设董事会）或双层制治理
   - 董事会可以自行决定分红标准，无需法定储备金
   - 可发行不同表决权股份（同股不同权），为创始人控制权设计提供便利

无直接税环境
   - 开曼不对公司征收企业所得税、资本利得税、预提税
   - 仅收取年度牌照费（Annual Fee），费用较低

中国企业的常见架构
   中国企业境外上市通常搭建**VIE架构**或**红筹架构**，以开曼公司作为
   上市主体：

   ::

    ┌────────────────────────────────────────────────┐
    │           开曼公司（上市主体）                   │
    │   ── 注册于开曼群岛、香港联交所/纳斯达克上市   │
    │   ── 通常采用A/B股架构，创始人持有B类超级      │
    │       表决权股，公众股东持有A类普通股           │
    └──────────────────┬─────────────────────────────┘
                       │ 100%控股
                       ▼
    ┌────────────────────────────────────────────────┐
    │         香港公司（中间控股层）                   │
    │   ── 作为中间控股公司，享受内地与香港税收安排  │
    │   ── 持有境内运营实体股权                      │
    └──────────────────┬─────────────────────────────┘
                       │ 100%控股
                       ▼
    ┌────────────────────────────────────────────────┐
    │       境内外商独资企业（WFOE）                  │
    │   ── 根据中国法律设立的有限责任公司             │
    │   ── 通过独家服务协议控制境内运营实体           │
    └────────────────────────────────────────────────┘

====
小结
====

不同法系下的公司制度各有优劣。大陆法系注重规则的体系化和小股东保护，
但在灵活性上有所欠缺；英美法系强调市场机制和契约自由，在创新激励方面
更具优势；以开曼为代表的离岸公司法则以"轻监管、重契约、零税收"为特色，
服务于跨境资本运作需求。

对中国企业而言，理解这些差异不仅是法律合规的需要，更是跨境经营和资本
运作的战略前提。
