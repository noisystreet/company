ESOP SPV架构图
==================

::

   创始人 → 有限合伙（SPV）
       ↑           ↓
     员工 ←    运营公司（预留15-20%期权池）

**典型结构**：

::

   运营公司
      │
      ├── 创始人（直接持股）
      ├── 机构投资者
      └── 员工持股平台（有限合伙SPV）
              │
              ├── GP（创始人/其控制公司 → 1%份额+100%管理权）
              └── LP（员工 → 99%份额，无管理权）

----------------------------------

结构详解
==============

**搭建流程**：

1. 运营公司董事会/股东会决议预留期权池（通常15-20%）
2. 创始人或指定主体设立有限合伙企业作为SPV
3. 运营公司向SPV增发股权/转让股权
4. 员工与SPV签署份额授予协议
5. 员工通过持有SPV份额间接享有运营公司收益

**管理方式**：

- **GP**（普通合伙人）：由创始人或其控制公司担任，负责SPV的管理决策
- **LP**（有限合伙人）：员工持有，享有经济收益但不参与SPV管理

----------------------------------

优点
========

1. **集中管理**——股权激励统一在一个平台中管理
2. **避免混乱**——员工离职时仅在SPV层面变更份额，不影响运营公司股东名册
3. **退出统一**——员工离职或被收购时，由SPV统一处理退出事宜
4. **控制权集中**——创始人作为GP，即使员工持有大量份额也不影响投票权
5. **税务效率**——有限合伙层面不缴企业所得税，穿透至员工个人纳税

----------------------------------

案例
========

**华为**：
- 采用工会持股（虚拟受限股）模式，员工通过工会间接持有公司股权
- 员工享有分红权和增值权，但无投票权

**字节跳动**：
- 设立有限合伙SPV作为员工持股平台
- 员工通过持有SPV份额间接持有字节跳动股权

中国科技公司vs美国科技公司ESOP对比
=========================================

.. list-table:: 中美ESOP对比
   :header-rows: 1

   * - 维度
     - 中国科技公司
     - 美国科技公司
   * - **上市架构**
     - VIE（受限于外资准入）
     - 直接持股（无外资限制）
   * - **AB股接受度**
     - 港股/科创板允许
     - 全面允许
   * - **期权税务**
     - 行权即纳税（现金压力大）
     - 出售时纳税
   * - **ESOP持有方式**
     - 有限合伙SPV间接持有
     - 直接授予RSU/NSO
   * - **创始人控制方式**
     - GP控制持股平台
     - AB股+同股不同权
   * - **典型上市耗时**
     - 融资4-8轮，6-10年
     - 融资3-6轮，5-8年
