===============================
各板块上市规则汇总
===============================

.. 本篇内容

   - 主板/科创板/创业板/北交所/新三板上市条件对比
   - 市值要求与利润要求
   - 板块选择建议

-------------------------------

上市条件总览
-------------------------------

各板块根据其定位不同，设置了差异化的上市条件。以下从**利润要求**和**交易门槛**两个核心维度进行对比：

.. list-table:: 各板块上市条件对比
   :header-rows: 1
   :widths: 15 20 20 25 20

   * - 板块
     - 定位
     - 利润要求
     - 市值要求
     - 投资者门槛
   * - **主板**
     - 大盘蓝筹
     - 近3年净利润均为正，累计≥1.5亿
     - 发行后股本总额≥5,000万
     - 无
   * - **科创板**
     - 硬科技
     - 允许未盈利
     - 预计市值≥40亿（未盈利情形）
     - 无
   * - **创业板**
     - 成长型创业
     - 近2年净利润均为正，累计≥5,000万
     - 发行后股本总额≥3,000万
     - 无
   * - **北交所**
     - 专精特新
     - 近1年净利润≥2,500万
     - 预计市值≥4亿
     - 无
   * - **新三板**
     - 中小微
     - 无（几乎零门槛）
     - 无要求
     - **500万金融资产**

-------------------------------

各板块详细上市条件
-------------------------------

主板
~~~~~~~~

- **净利润要求**：最近3个会计年度净利润均为正数，且累计不低于人民币1.5亿元
- **市值及财务指标**：最近3个会计年度经营活动产生的现金流量净额累计不低于人民币1亿元；或最近3个会计年度营业收入累计不低于人民币10亿元
- **股本要求**：发行后股本总额不低于人民币5,000万元

科创板
~~~~~~~~

- **市值≥40亿**：主要产品或服务经国家有关部门批准，市场空间大，目前已取得阶段性成果；医药行业企业至少有一项核心产品获准开展II期临床试验
- **市值≥30亿**：最近一年营业收入不低于人民币3亿元
- **允许未盈利**：科创板是唯一允许未盈利企业在A股上市的板块（需满足相应市值标准）

创业板
~~~~~~~~

- **一般标准**：最近2年净利润均为正，且累计净利润不低于人民币5,000万元
- **市值标准**：市值≥50亿且最近一年营收≥3亿（适用于未盈利情形，但改革后已收紧）
- **发行后股本**：不低于3,000万元

北交所
~~~~~~~~

- **标准一**：预计市值≥2亿，最近2年净利润均≥1,500万元，加权平均净资产收益率平均≥8%；或最近1年净利润≥2,500万元且加权平均净资产收益率≥8%
- **标准二**：预计市值≥4亿，最近1年营收≥1亿，且最近1年经营活动现金流量净额为正
- 北交所定位于"专精特新"中小企业，是连接新三板与沪深交易所的重要枢纽

新三板
~~~~~~~~

- **基础层**：无利润要求，几乎零门槛
- **创新层**：有成长性要求（如最近2年营收增长≥30%或最近1年净利润≥800万等）
- **精选层**（现已整体平移至北交所）：有利润要求的挂牌标准

.. note::

   新三板本质上不同于A股上市——它是"挂牌"而非"上市"，
   因此门槛远低于沪深交易所和北交所。

-------------------------------

板块选择建议
-------------------------------

企业在选择上市板块时，应综合考虑以下因素：

::

   1. 行业属性：
      - 传统行业、大型企业 → 主板
      - 硬科技、半导体、生物医药 → 科创板
      - 成长型创新创业 → 创业板
      - 专精特新中小企业 → 北交所
      - 暂未达标的中小微企业 → 新三板（先挂牌培育）

   2. 盈利状况：
      - 持续盈利能力强 → 主板 / 创业板
      - 尚未盈利但技术领先 → 科创板
      - 盈利规模较小但有成长性 → 北交所

   3. 融资需求：
      - 大规模融资 → 主板 / 科创板 / 创业板
      - 中小规模融资 → 北交所 / 新三板

   4. 时间规划：
      - 急于上市 → 北交所（审核周期较短）
      - 愿意等待 → 主板 / 科创板 / 创业板（审核流程更长但融资规模更大）

.. tip::

   企业可采取"先新三板挂牌→转层→转板北交所→再转沪深"的**递进式上市路径**，
   但需注意转板率极低，实际执行难度较大。
