2.4.1. 董事会成员组成

2.4.1.1. 谁可以当董事？

任何**具有完全民事行为能力的自然人**都可以当董事。但有以下限制：

董事资格限制

限制情形

说明

无/限制民事行为能力人

❌ 精神疾病、未成年人

有经济犯罪前科（未满5年）

❌ 贪污/贿赂/侵占财产等

被证监会市场禁入（未满）

❌ 证券市场禁入人员

失信被执行人

❌ 老赖

公务员/现役军人（兼职限制）

❌ 体制内人员不得兼任企业董事

不限国籍、不限持股——不一定要是股东才能当董事。

2.4.1.2. 董事的三种类型

董事类型

类型

谁担任

是否全职

作用

执行董事（Executive Director）

公司内部高管（CEO/CTO/CFO等）

✅ 全职在公司上班

参与决策+执行落地

非执行董事（Non-Executive Director）

投资人代表、外部专家

❌ 兼职（只来开会）

代表投资方利益、提供行业建议

独立非执行董事（Independent Director）

与公司无任何利益关联的第三方

❌ 兼职

监督制衡、保护中小股东

某B轮AI芯片公司的7人董事会：

  执行董事（3人）：
    ├── 创始人张总（CEO）── 全职，管公司日常
    ├── 李总（CTO）      ── 全职，管技术
    └── 王总（CFO）      ── 全职，管财务

  非执行董事（2人）：
    ├── 红杉资本代表     ── 投资人，每季度来开一次会
    └── 高瓴资本代表     ── 投资人，同上

  独立董事（2人）：
    ├── 某大学教授       ── 行业专家，独立判断
    └── 某退休会计师     ── 财务专家，审财报

2.4.1.4. 独立董事的特殊要求（上市公司）

港交所/上交所对独立董事有"独立性测试":

以下人员不能当独立董事：
  ❌ 在公司/子公司任职的人
  ❌ 与公司有业务往来的律所/会所的合伙人
  ❌ 公司董事/高管的近亲属
  ❌ 持有公司股份超过1%的人
  ❌ 在公司任职不足5年的人

独立董事占比要求：通常≥董事会人数的1/3，审计委员会必须全部由独立董事组成。

2.4.1.5. 独立董事选举流程

2.4.1.5.1. 未上市公司

创始人（控制董事会）→ 提名一位行业权威人士 → 董事会投票通过 → 双方都觉得靠谱
投资人（有董事会席位）→ 提名一位财务专家 → 董事会投票通过 → 代表投资方利益

实际情况通常是创始人和投资人在投资协议中提前约定：
  "独立董事候选人须经创始人和领投方共同同意。"

未上市公司的独立董事本质上是**双方妥协的产物**——既不能是创始人的人，也不能是投资人的人。

2.4.1.5.2. 上市公司

流程复杂得多:

上市公司独立董事选举流程：

第一步：① 董事会/监事会/持股1%以上股东 → 提名候选人
       ② 上交所/港交所审核独立性 → 通过后才能成为正式候选人

       ④ 年度股东大会 → 全体股东投票（一股一票）

第三步：⑤ 得票过半 → 正式成为独立董事
       ⑥ 签独立性声明 → 报交易所备案

2.4.3. 董事会怎么选出来的？

核心原则：董事由**股东会选举**产生，每届任期**不超过3年**，可连选连任。

**选举流程**（未上市公司）:

第一步：提名
  创始人/大股东/投资人 → 提出董事候选人名单
  （通常在投资协议中已提前约定好各方的席位分配）

第二步：股东会表决
  股东会召开 → 全体股东投票（一股一票）
  候选人获半数以上表决权通过 → 当选

第三步：备案
  选出的董事在工商系统备案登记

**上市公司**的流程更复杂:

第一步：提名
  董事会/监事会/持股3%以上股东（A股）/持股1%以上股东（港股）
  → 提出董事候选人名单

第二步：审核
  交易所核查候选人资格（特别是独立董事的独立性）
  → 通过后才能成为正式候选人

第三步：股东大会投票
  年度股东大会召开
  → 全体股东投票（一股一票）
  → 得票过半 → 正式成为董事

第四步：公告
  选举结果公告 → 交易所备案

2.4.3.1. 累积投票制（保护小股东）

这是一个能让小股东把自己所有票集中投给一个人的制度:

普通投票制（一股一票）：
  大股东持70% → 70票 → 他可以7个名额各投10票 → 全部是他的人
  小股东持30% → 30票 → 分散投票 → 一个名额都拿不到
  → 董事会全是大股东的人

累积投票制：
  小股东持30%，有7个名额 → 30×7=210票 → 全部投给1个人
  大股东持70%，有7个名额 → 70×7=490票 → 需要分配给7个人
  → 小股东集中火力，至少可以保证**1个名额**当选
  → 这就是"保护少数股东"的核心机制
累积投票制的适用范围

说明

✅ 强制

未上市公司

❌ 可选

章程中可以约定是否采用，多数不采用

港股/美股

❌ 不强制

2.4.3.3. 常见问题"董事会能选董事会吗？"

不能。董事会的权力是管理公司，不是选举自己。

正确逻辑链：
  股东 → 选举 → 董事（进入董事会）
  董事 → 投票 → 董事长（董事会内部选举产生）
  董事会 → 聘任 → CEO/总经理（管理层）

错误理解：
  董事会 → 选举 → 新董事 ❌（这是股东会的权力）

2.4.3.4. 未上市公司与上市公司的选举本质差异

关键认知：未上市公司的董事会席位**不是在股东会上"选"出来的，而是在投资协议里"谈"出来的。**:

时间线：

融资谈判阶段（签SHA/股东协议）：
  "A轮投资人占1个董事会席位"
  "B轮投资人占1个董事会席位"
  "创始人占2个董事会席位"

  投资人B（投了票）："没意见。"
  → 表决通过 → 五人正式成为董事

2.4.4. 董事会席位分配（轮次制）

2.4.4.1. 核心原则：按"轮次"分，不是按"家数"

错误做法（没人这样分）：
  A轮红杉要1席 + B轮高瓴要1席 + C轮腾讯要1席 + C轮阿里要1席 + C轮深创投也要1席...
  → 董事会膨胀到10人以上，没法干活

正确做法：
  A轮投资人：1席（代表）
  B轮投资人：1席（代表）
  C轮投资人：1席（代表）
  → 无论C轮有5家还是10家投资机构，合起来只占1席

2.4.4.2. 领投方席位制——具体怎么分

假设公司融资到C轮，共15家投资机构:

C轮（5家机构，合计投8亿）：
  腾讯（领投，投4亿）：C轮董事会席位 ✅
  阿里（跟投，投2亿）： 无席位
  红杉（跟投，投1亿）： 无席位
  高瓴（跟投，投0.5亿）：无席位
  深创投（跟投，投0.5亿）：无席位
  → C轮5家机构共享1个董事会席位，由领投方（腾讯）派出代表

2.4.5. 董事会僵局（Deadlock）

当创始人和投资人席位相等（如2:2），且独立董事无法形成决定性意见时，董事会就会陷入**僵局（Deadlock）**。

僵局的具体表现:

董事会投票结果：
  创始人阵营： 2票赞成
  投资人阵营： 2票反对
  → 赞成票 ≠ 反对票？不对，是 2:2 平局
  → 决议**无法通过**
  → 会议陷入僵局，下次再议还是2:2
谁有最终决定权？

场景

处理方式

有独立董事

独立董事一票定乾坤——他是3:2的关键一票

无独立董事，且2:2平局

🔴 僵局——会议无果而终

CEO兼任董事长有额外投票权

**董事长可投两次**（少见，需章程约定）

章程约定"董事长有终裁权"

**董事长可以拍板**（需提前写在章程里）

做法

备注

为什么"提交股东会"就能解决？ 因为股东会是公司的最高权力机构。 董事会解决不了的争议，上升到股东会由全体股东投票——谁持股多谁说了算。 在未上市公司中，创始人通常有GP-LP或AB股保护，股东会反而比董事会更容易控制。 这就是为什么创始人更倾向于把争议提到股东会，而投资人更倾向于在董事会层面解决。

决议类型与僵局

处理方式

影响

普通决议（日常经营）
