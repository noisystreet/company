9.4. 控制权争夺案例

9.4.1. 比特大陆：吴忌寒 vs 詹克团（2019）

背景：比特大陆是全球最大比特币矿机生产商。两位创始人：吴忌寒（负责商业/金融）vs 詹克团（负责技术/产品）。股权相近，董事会各有支持者。

9.4.1.1. 比特大陆案经过

案例：比特大陆案经过

2019年10月，吴忌寒趁詹克团出差深圳参加安博会， 在北京发动"闪电政变"：

第一步：凭VIE架构控制权，变更北京公司的法定代表人和执行董事 第二步：发全员邮件宣布解除詹克团一切职务，"即刻生效" 第三步：解除詹克团招来的HR负责人，换回自己人 第四步：召开员工大会，历数詹克团决策失误（10nm芯片流片损失15亿美元） 第五步：封锁公司权限，詹克团回京后连公司都进不去

詹克团反应：沉默10天后发公开信
"当巨变来临的时候我才知道，被自己最信任的合作伙伴背后捅刀的桥段，

真的会发生。"

9.4.1.2. 控制权争夺手段

比特大陆控制权争夺手段

手段

说明

VIE架构控制

利用境外VIE架构对境内实体的控制权，变更法定代表人和执行董事

闪电通知

在对方出差不在场时发动，造成既成事实

人事清洗

第一时间更换HR负责人，巩固控制

员工大会

争取员工支持，制造舆论压力

9.4.1.3. 结局

案例：比特大陆案结局

结局：两位创始人分家——比特大陆拆分，各拿一部分业务继续运营

9.4.1.4. 比特大陆案教训总结

比特大陆案教训

教训

分析

双创始人风险

没有AB股保护的"双创始人"结构，一旦翻脸就是毁灭性的

速度决定一切

即使公司估值百亿，董事会政变可以在一个下午完成

VIE架构的双刃剑

可以成为控制工具，也可能被对方利用

9.4.2. 雷士照明：吴长江三进三出（2005-2016）

背景：吴长江是中国照明行业最传奇也最悲情的创始人。他创立的雷士照明在16年内三次被赶出自己创办的公司。

9.4.2.1. 雷士照明案经过

案例：雷士照明案经过

第一次出局（2005年）—— 被合伙人赶走：

杜刚、胡永宏合计持股55%，在董事会联合发难 → 罢免吴长江的董事长职务（董事会多数即可） → 吴长江被迫同意以8000万元转让全部股权退出 → 经销商集体罢工抗议 → 吴长江花1.6亿买回全部股份重掌公司

第二次出局（2012年）—— 被资本赶走：

软银赛富+施耐德在董事会占多数席位 → 第一步：董事会罢免吴长江董事长职务 ✅ → 第二步：直接"连锅端"——吴长江辞去全部职务（CEO+董事） → 软银赛富阎焱接任董事长，施耐德指派CEO → 经销商+员工+供应商全面罢工抗议 → 吴长江艰难复职，但持股已不足20%

第三次出局（2014年）—— 彻底终结：

资本方已完全控制董事会 → 董事会投票罢免CEO职务 ✅ → 股东会投票罢免董事身份 ✅ → 持股已被稀释到极低，无力回天 → 2016年吴长江因挪用资金被判刑14年

9.4.5. 周总温水煮青蛙

9.4.5.1. 周总案经过

案例：周总案经过

周总（化名），汽车零部件公司，年营收近10亿：

第一轮融资（投1亿，占20%）：

签了投资人要求的条款： - 投资人有权委派1名董事（董事会5席：创始人2+投资人1+独董2） - 重大事项需2/3以上董事同意 - 投资人对特定事项有一票否决权 结果：创始人还能控制董事会

第二轮融资（投2亿，占15%）：

新投资人要求扩大董事会：5席→7席 新分配：创始人2席 vs 投资人4席(3轮投资各1席+1席) vs 独董1席 结果：创始人只剩2/7，重大事项必须求投资人同意

触发点：创始人想卖掉亏损子公司做上市准备

投资人：行使一票否决权 → 不批准 创始人：已经做不了任何他想做的事了
