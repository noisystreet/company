======================
03　公司制度的核心架构
======================

.. _chapter-01-architecture:

==============
概述：三会一层
==============

现代公司治理的核心是**"三会一层"**架构——即股东会（权力机构）、董事会
（决策机构）、监事会（监督机构）和管理层（执行机构）的分权制衡体系。这
一架构根植于公司所有权与经营权分离的基本前提，旨在解决委托代理问题。

下图为三会一层的治理架构关系：

::

   ┌─────────────────────────────────────────────────────┐
   │                   股东会（权力机构）                   │
   │   ── 最高权力机关，决定公司重大事项                  │
   │   ── 选举/更换董事、监事                            │
   │   ── 审议批准董事会和监事会报告                      │
   │   ── 修改公司章程，增减注册资本                      │
   └────────────┬────────────────────┬──────────────────┘
                │ 选举并监督         │ 选举并监督
                ▼                    ▼
   ┌────────────────────┐  ┌──────────────────────┐
   │  董事会（决策机构）  │  │  监事会（监督机构）    │
   │  ── 经营决策机构    │  │  ── 监督董事和高管   │
   │  ── 聘任/解聘高管   │  │  ── 检查公司财务     │
   │  ── 制定经营计划    │  │  ── 列席董事会会议   │

   │   ── 执行董事会决议，负责日常运营                    │
   │   ── 主持生产经营管理工作                            │

================
各机构的权力配置
================

==================
股东会（股东大会）
==================

股东会是公司的**最高权力机构**，由全体股东组成。股东会通过行使表决权
对公司重大事项作出决议。根据中国公司法（2024修订），股东会行使以下职权：

- 选举和更换董事、监事，决定其报酬事项
- 审议批准董事会的报告
- 审议批准监事会的报告
- 审议批准公司的年度财务预算方案、决算方案
- 审议批准公司的利润分配方案和弥补亏损方案
- 对公司增加或者减少注册资本作出决议
- 对发行公司债券作出决议
- 对公司合并、分立、解散、清算或者变更公司形式作出决议
- 修改公司章程

股东会决议需经代表**过半数表决权**的股东通过（普通决议），修改章程、
增减资本等重大事项需经**三分之二以上**表决权通过（特别决议）。

======
董事会
======

董事会是公司的**经营决策机构**，对股东会负责。董事会成员由股东会选举
产生。中国公司法规定有限责任公司董事会成员为**三人以上**。

董事会的主要职权包括：

- 召集股东会会议，并向股东会报告工作
- 执行股东会的决议
- 决定公司的经营计划和投资方案
- 制订公司的年度财务预算方案、决算方案
- 制订公司的利润分配方案和弥补亏损方案
- 制订公司增加或者减少注册资本以及发行公司债券的方案
- 制订公司合并、分立、解散或者变更公司形式的方案
- 决定公司内部管理机构的设置
- 聘任或者解聘公司经理及其报酬事项
- 制定公司的基本管理制度

2024年新公司法强化了**董事的忠实义务和勤勉义务**，并明确规定了董事对
第三人的损害赔偿责任，显著提升了董事的履职风险。

======
监事会
======

监事会是公司的**监督机构**，对董事、高级管理人员的职务行为进行监督。
监事会成员不得少于**三人**，应当包括股东代表和适当比例的公司职工代表。

监事会的职权包括：

- 检查公司财务
- 对董事、高级管理人员执行职务的行为进行监督
- 当董事、高级管理人员的行为损害公司利益时，要求其予以纠正
- 提议召开临时股东会会议
- 向股东会会议提出提案
- 依法对董事、高级管理人员提起诉讼

2024年新公司法引入了**审计委员会制度**——公司可以在董事会下设审计
委员会替代监事会，为治理结构提供了更多灵活性。

======
管理层
======

管理层由总经理、副总经理、财务负责人等高级管理人员组成，是公司的**执行
机构**。管理层由董事会聘任，对董事会负责。

管理层的主要职责：

- 主持公司的生产经营管理工作
- 组织实施董事会决议
- 组织实施公司年度经营计划和投资方案
- 拟订公司内部管理机构设置方案
- 拟订公司的基本管理制度
- 制定公司的具体规章

==============
权力制衡的逻辑
==============

三会一层的设计体现了**"决策权—执行权—监督权"三权分立**的治理哲学：

.. list-table:: 三权分立的制衡逻辑
   :widths: 15 15 25 25 20
   :header-rows: 1

   * - 权力类型
     - 行使主体
     - 核心职责
     - 制衡对象
     - 制衡机制
   * - 所有权（决策权）
     - 股东会
     - 重大事项决定
     - 董事会越权行为
     - 审议批准+任免
   * - 经营权（执行权）
     - 董事会+管理层
     - 日常经营决策与执行
     - 股东会不当干预
     - 法定权限边界
   * - 监督权
     - 监事会
     - 财务与合规监督
     - 权力滥用与渎职
     - 检查+纠正+诉讼

这一制衡体系的核心逻辑在于：**任何一方都不能独自控制公司的全部权力**。
股东会拥有最终决定权但缺乏日常参与的能力；董事会拥有经营决策权但受
股东会约束和监事会监督；监事会拥有监督权但无权干预经营。三方在各自
的权限范围内行使权力，相互制约，共同维护公司的健康发展。

======================
2024年新法下的治理变革
======================

2024年新公司法对三会一层架构进行了重要调整：

- **审计委员会替代监事会**：允许公司选择在董事会下设审计委员会来履行
  监事会职责，这是治理结构的重大创新
- **董事责任强化**：引入董事对第三人责任条款，董事在执行职务时因故意
  或重大过失给他人造成损害的，应当承担赔偿责任
- **小股东保护升级**：扩大股东查阅权范围，降低股东提案权的持股比例
  要求，强化了控股股东的义务

这些变化表明，中国公司治理正朝着**更加灵活、更加严格、更加注重中小
股东保护**的方向演进。
