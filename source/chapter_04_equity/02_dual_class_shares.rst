==================
AB股（同股不同权）
==================

----------------------------------

AB股概述
------------

**典型比例**：A类股1:1投票权，B类股1:10-20投票权

- **A类股**：公众股东/外部投资人持有，每份1票
- **B类股**：创始人/核心团队持有，每份10-20票
- **目的**：融资到只剩15%股权时，仍控制80%投票权

经典案例
------------

.. list-table:: AB股经典案例
   :header-rows: 1

   * - 公司
     - 创始人
     - 超级投票权倍数
     - 投票权占比
   * - 小米
     - 雷军
     - B类股1:10
     - ~55%
   * - 蔚来
     - 李斌
     - C类股1:8
     - ~44%
   * - 美团
     - 王兴
     - A类股1:10
     - ~47%
   * - 京东
     - 刘强东
     - B类股1:20
     - ~70%

**京东案例换算逻辑**：

刘强东持~12%股权控制70%投票权：

::

   总投票权 = (12% × 20) + (88% × 1) = 328%
   创始人投票权占比 = 240% ÷ 328% ≈ 73%

----------------------------------

投资人为什么接受AB股
------------------------

投资人虽然投票权少，但看重的是：

1. **创始人能力**——创始人对公司方向的理解比机构更深刻
2. **长期主义**——防止短期机构为了财报好看而逼迫创始人做错误决策
3. **赚钱不受影响**——AB股只影响投票权，不影响股权的经济回报（分红、出售收益都一样）

----------------------------------

AB股风险
------------

.. list-table:: AB股主要风险
   :header-rows: 1

   * - 风险
     - 说明
   * - 创始人失控
     - 创始人能力退化后，无人能更换CEO
   * - 估值折价
     - 部分机构因治理风险要求估值打折
   * - 日落条款
     - 港交所要求B类股在创始人离职/去世后自动转为A类股

----------------------------------

各市场规则
--------------

.. list-table:: 各市场AB股规则
   :header-rows: 1

   * - 市场
     - 是否允许
     - 说明
   * - 美股（纳斯达克/NYSE）
     - ✅ 完全允许

   * - 港股（港交所）

     - ✅ 允许

   * - A股主板
     - ❌ 不允许
     - 必须同股同权

----------------------------------

方案一：开曼公司架构（美股/港股通用）
--------------------------------------------

中概股标准做法。公司注册在开曼群岛，上市主体也是开曼公司。

::

   开曼公司（上市主体）
    ├── 普通股（Ordinary Shares）→ 创始人拿
    ├── A类普通股 → 公众股东（IPO后拿，1:1投票权）
    └── B类普通股 → 创始人拿（1:10-20投票权）

.. list-table:: 开曼架构实施步骤
   :header-rows: 1

   * - 步骤
     - 内容
   * - ① 开曼注册
     - 在开曼注册一家豁免公司
   * - ② 章程
     - 在《公司章程》中写明AB股条款
   * - ③ 分配
     - 创始人拿B类股，投资人拿A类普通股或优先股
   * - ④ IPO
     - 向港交所/纳斯达克提交招股书，披露AB股结构
   * - ⑤ 上市后
     - A类股在交易所公开交易，B类股不交易（只由创始人持有）

**典型公司**：小米、美团、蔚来、京东

**成本**：开曼公司注册+年费约2-5万元/年，整个VIE架构搭建成本约20-50万元

----------------------------------

方案二：A股科创板特殊表决权
----------------------------------

2018年改革后，科创板允许设置"特别表决权"。

::

   境内股份公司（科创板）
    ├── 普通股份（每份1票）→ 公众股东
    └── 特别表决权股份（每份X票，上限10票）→ 创始人

.. list-table:: 科创板特殊表决权限制
   :header-rows: 1

   * - 限制
     - 规则
   * - 倍数上限
     - 特别表决权≤普通股10倍
   * - 持股下限
     - 持特别表决权的股东合计持股≥10%
   * - 市值要求
     - 预计市值≥100亿元，或≥50亿元+营收≥5亿
   * - 上市后
     - 特别表决权股份不得在二级市场交易

**局限**：门槛极高（100亿市值），不适合早期公司。

----------------------------------

方案三：境内有限公司变通方案（未上市阶段最常用）
--------------------------------------------------------

未上市境内公司无法直接做AB股，但可通过以下方式实现类似效果：

方式A：一致行动人协议
~~~~~~~~~~~~~~~~~~~~~~~~

::

   创始人（20%）+ 联合创始人（10%）+ 老员工（5%）
            ↓ 签署《一致行动人协议》
            ↓ 三个人在股东会投票时必须保持一致
            ↓ 创始人通过协议控制35%的投票权

- **优点**：简单，成本低（几千元律师费）
- **缺点**：协议可撕毁，法律约束力弱

方式B：GP-LP有限合伙架构（最常用）
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

::

   创始人只有1%资金，但做GP（普通合伙人），100%管理权。
   投资人有99%资金，但做LP（有限合伙人），0%管理权。

       创始人（GP，1%）         投资人/员工（LP，99%）
            \                        /
             ↓                      ↓
           持股平台（有限合伙）100%持有运营公司股权

- 这是中国科技公司未上市阶段最常见的"隐形AB股"
- 不需要改公司章程，通过合伙协议就能实现
- **案例**：昉擎科技——梁军通过多个GP壳控制33.84%股权但管理权远超

方式C：公司章程特殊约定（2024新公司法）
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

2024年新公司法允许有限公司在章程中约定差异化表决权：

::

   有限公司章程中可以写：
     "股东A所持每元出资额对应5票表决权；
     股东B所持每元出资额对应1票表决权。"

- 实践中使用较少，工商局备案可能遇到障碍

----------------------------------

三种实现方案对比
--------------------

.. list-table:: 三种AB股实现方案对比
   :header-rows: 1

   * - 维度
     - 开曼架构（美股/港股）
     - 科创板（A股）
     - 境内有限公司
   * - **AB股倍数**
     - 最高20倍
     - 最高10倍
     - GP-LP可实现"无限倍"
   * - **适用阶段**
     - IPO前+上市后
     - 仅上市后
     - 融资全阶段
   * - **成本**
     - 高（20-50万搭建）
     - 中等（上市辅导费）
     - 低（几千-几万）
   * - **灵活性**
     - 极高
     - 严格受限
     - 高
   * - **适用范围**
     - 准备境外上市的公司
     - 已达科创板上市门槛
     - 所有未上市科技公司

----------------------------------

实操建议
------------

::

   早期（天使-A轮）：用GP-LP架构或一致行动人协议 → 成本低，够用
   中期（B轮以后）：搭建开曼VIE架构 → 为境外上市准备
   上市阶段：章程写入AB股条款 → 最终锁定控制权

.. important::

   **关键时间点**：B轮融资之前，必须把控制权结构设计好。B轮以后机构已经是大股东，再想改成AB股，需要股东会通过——机构不会同意。
