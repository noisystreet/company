========
估值方法
========

.. 本篇内容

   - 可比公司法（Comps）
   - 现金流折现法（DCF）
   - 历史交易法
   - 各方法适用场景对比
   - 早期公司常见估值陷阱

----------------------------------

估值是融资过程中最具争议性的核心议题。对投资人而言，估值决定了投资回报率；对创始人而言，估值决定了出让股权的成本。估值过高可能导致下轮融资困难，估值过低则过度稀释创始团队。

----------------------------------

三类基础估值方法
================

① 可比公司法（Comparable Company Analysis）
-------------------------------------------

**原理**：寻找与目标公司业务模式、阶段、规模相近的已上市公司或近期融资公司，以其估值倍数作为参照。

::

   计算公式（以PS倍数为范例）：

     目标公司估值 = 目标公司营收 × 可比公司PS倍数中位数

   示例：
     可比公司平均PS倍数为5-8x
     目标公司年营收5000万
     ⇒ 估值区间约2.5亿-4亿人民币

**常用估值倍数**：

.. list-table:: 常见估值倍数及应用场景
   :header-rows: 1

   * - 倍数类型
     - 适用行业
     - 说明
   * - **P/E（市盈率）**
     - 盈利稳定的成熟企业
     - 最常用，但不适于亏损企业
   * - **P/S（市销率）**
     - 高成长、尚未盈利的科技公司
     - SaaS公司通常3-15x PS
   * - **EV/EBITDA**
     - 重资产、高折旧行业
     - 消除折旧和资本结构的影响
   * - **EV/Revenue**
     - 早期亏损公司
     - 更纯粹的营收倍数
   * - **GMV倍率**
     - 平台型企业
     - 关注交易规模而非收入

**优缺点**：

::

   优点：
     - 简单直观，沟通成本低
     - 反映市场当前偏好

   缺点：
     - 难以找到真正可比的标的
     - 受市场情绪影响大（牛市估值虚高）
     - 忽视公司间差异化因素

----------------------------------

② 现金流折现法（DCF - Discounted Cash Flow）
--------------------------------------------

**原理**：预测公司未来自由现金流，以适当折现率将其折现到现值。

::

   基本公式：

       公司价值 = Σ [FCFt / (1 + r)^t] + 终值

       其中：
         FCFt = 第t年的自由现金流
         r    = 折现率（WACC，加权平均资本成本）
         终值   = 预测期后的持续价值（通常占60-80%）

   五步法流程：

   第一步：预测未来5-10年财务数据
   第二步：估算自由现金流
   第三步：确定折现率（WACC）
   第四步：计算预测期现值
   第五步：计算终值并加总

**优缺点**：

::

   优点：
     - 理论上最严谨的方法
     - 反映公司内在价值而非市场情绪

   缺点：
     - 对早期公司预测极其困难
     - 结果对假设（增长率、折现率、终值）高度敏感
     - 终值占比过高，削弱了预测的可信度

**敏感性分析示例**：

.. list-table:: DCF敏感性分析（估值单位：亿元）
   :header-rows: 1

   * - 永续增长率
     - WACC=10%
     - WACC=12%
     - WACC=15%
   * - 2%
     - 35.2
     - 22.8
     - 14.3
   * - 3%
     - 42.6
     - 26.9
     - 16.5
   * - 4%
     - 54.1
     - 32.7
     - 19.4

   可见，假设参数的微小差异会导致估值结果大幅变化。

----------------------------------

③ 历史交易法（Precedent Transaction）
-------------------------------------

**原理**：参考同行业公司被收购或融资的历史交易价格，作为估值锚定。

::

   典型应用场景：
     - 并购交易定价
     - 私有化退市定价
     - 二级市场定增定价

   注意事项：
     - 交易溢价通常包含控制权溢价（15-30%）
     - 需将交易时点与当前市场环境做调整
     - 历史交易样本通常有限，统计意义较弱

估值方法适用场景对比
====================

.. list-table:: 三种估值方法适用场景对比表
   :header-rows: 1
   :widths: 15 25 25 35

   * - 方法
     - 最佳适用场景
     - 弱处
     - 早期公司可用性
   * - **可比公司法**
     - 已上市的行业对标、稳定盈利
     - 缺乏对标标的、非标准化业务
     - ★★★☆☆ 可作为参考锚点
   * - **现金流折现法**
     - 成熟期、现金流可预测
     - 早期亏损、增长路径不确定
     - ★☆☆☆☆ 实用性有限
   * - **历史交易法**
     - 并购定价、特定赛道估值
     - 样本少、交易特征差异大
     - ★★☆☆☆ 可用但需谨慎

早期公司常见估值陷阱
====================

① 融资驱动型估值过高
--------------------

::

   问题：创始人为少稀释股权而追求过高估值
   后果：
     - 下轮融资估值倒挂（Down Round）
     - 触发反稀释条款，导致创始人被大幅稀释
     - 员工期权行权价过高，失去激励效果
   建议：估值应基于业务实质，而非融资需求

② 忽视"有效估值"
----------------

::

   有效估值 = 名义估值 - 优先权价值 - 期权池调整

   很多创始人只看名义Pre-money估值，忽略了：
     - 清算优先权（1x优先+参与分配 = 实际估值打折扣）
     - 期权池（通常要求预留15-20%，这部分实际由创始人承担）
     - 对赌条款（若未达标，估值可能被下调）

③ 混淆"估值"与"出价"
--------------------

::

   投资人的出价 = 估值 × 投资金额
   但估值≠公司价值，仅是投资人愿意接受的入股价格

   关键区别：
     - 二级市场：价格由供需决定
     - 一级市场：价格由一对多谈判决定
     - 低出价 ≠ 公司价值低，可能是投资人风险偏好低

④ 忽略"估值逻辑"的一致性
------------------------

::

   错误示范：
     - 用DCF算远期价值，又用PS倍数定当下价格
     - 拿成熟公司的PE倍数套用亏损的初创公司
     - 忽视行业周期对估值倍数的影响

   正确做法：
     - 明确估值目的（融资？收购？内部激励？）
     - 选择与该目的匹配的估值方法
     - 交叉验证：至少使用2-3种方法做区间估算

⑤ 控制权对估值的影响
--------------------

::

   相同公司、相同阶段，不同控制权安排可能导致不同估值：

   出让20%具有超投票权的普通股        → 估值溢价更高
   出让同等投票权的优先股              → 估值折价更大
   带董事会否决权的投资者              → 通常要求更低的估值
