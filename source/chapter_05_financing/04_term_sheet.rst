==============================
投资条款（Term Sheet）核心要素
==============================

.. 本篇内容

   - Term Sheet结构与核心条款
   - 关键条款详解（清算优先权、防稀释、对赌等）
   - 创始人最应关注的条款排名

----------------------------------

Term Sheet（条款清单/投资意向书）是投资人与创业公司达成初步意向后签署的非约束性文件，它框定了融资交易的核心条款。虽然TS本身不具法律约束力，但其"排他性条款"和"保密条款"通常具有法律效力，且后续正式协议（SPA/SHA）基本沿袭TS框架。

----------------------------------

TS核心条款解析
==============

① 估值与投资额
--------------

::

   关键参数：
     - Pre-money估值     = 投资前公司价值
     - 投资金额           = 投资人投入资金
     - Post-money估值    = Pre-money + 投资金额
     - 投资人持股比例    = 投资金额 / Post-money估值

   示例：
     Pre-money估值：8000万
     投资金额：2000万
     Post-money估值：1亿
     投资人持股：20%

   陷阱提示：
     - "投前估值未包含期权池"（即创始人实际承担的期权成本）
     - 隐性费用（尽调费、顾问费要从投资额中扣）

----------------------------------

② 清算优先权（Liquidation Preference）
--------------------------------------

**定义**：当公司发生清算事件（出售、合并、破产）时，投资人优先于普通股股东获得分配的权利。

.. list-table:: 清算优先权类型对比
   :header-rows: 1

   * - 类型
     - 描述
     - 对公司影响
   * - **1x 非参与分配**
     - 投资人收回投资本金（1倍），剩余按持股比例分配
     - 对创始人最友好
   * - **1x 参与分配（Full Participation）**
     - 投资人收回1倍本金后，再按持股参与剩余分配
     - 投资人双重获利，创始人被动
   * - **1x 带上限参与分配（Capped Participation）**
     - 参与分配但有上限（通常2-3倍）
     - 折中方案
   * - **多倍优先（2x/3x）**
     - 投资人优先收回2-3倍投资额
     - 极度不利于创始人和员工

**谈判要点**：创始人应力争1x非参与分配，至少是带上限的参与分配。

----------------------------------

③ 防稀释条款（Anti-Dilution）
-----------------------------

**定义**：当公司后续以更低估值融资（Down Round）时，保护投资人持股比例的机制。

::

   加权平均 vs 完全棘轮：

   ① 完全棘轮（Full Ratchet）—— 对创始人最严苛
      → 投资人将转换价格调整至新股发行价
      → 无论新股发行多少，都照此调整

   ② 加权平均（Weighted Average）—— 相对公允
      → 考虑新股发行量和发行价，加权计算新转换价
      → 标准公式：

        新转换价 = 旧转换价 × (旧股数 + 新股数)
                               ─────────────────
                              (旧股数 + 新股数 × 新股价 ÷ 旧转换价)

**谈判要点**：
- 创始人应坚决拒绝完全棘轮
- 加权平均法是行业标准，宽基加权平均比窄基更友好

----------------------------------

④ 董事会席位（Board Composition）
---------------------------------

.. list-table:: 董事会席位安排方案对比
   :header-rows: 1

   * - 方案
     - 构成
     - 特点
   * - **3人董事会**
     - 创始人2席 + 投资人1席
     - 创始人控制，适合早期
   * - **5人董事会**
     - 创始人2席 + 投资人2席 + 独立董事1席
     - 制衡结构，适合A-B轮
   * - **7人董事会**
     - 创始人3席 + 投资人3席 + 独立董事1席
     - 融资后期常见结构

**关键条款**：投资人通常要求"特定事项否决权"（Protective Provisions），即虽然投资人不占多数席位，但特定事项（增资、并购、修改章程等）需经投资人董事同意。

----------------------------------

⑤ 保护性条款（Protective Provisions）
-------------------------------------

::

   投资人要求的一票否决权事项通常包括：

   □ 修改公司章程（尤其是股权相关条款）
   □ 增发新股或进行股权融资
   □ 合并、收购、出售重大资产
   □ 清算或解散公司
   □ 变更主营业务
   □ 单笔超过特定金额的借款或担保
   □ 支付超过特定金额的董事薪酬
   □ 与关联方进行重大交易

   创始人谈判策略：
     - 设置金额门槛（如500万以上才需同意）
     - 要求"大多数投资人同意"而非"任一投资人同意"
     - 随着融资轮次推进逐步取消早期投资人的单项否决权

----------------------------------

⑥ 对赌条款（Valuation Adjustment Mechanism / VAM）
--------------------------------------------------

参考公司治理手册中关于":ref:`对赌条款 <chapter-03-valuation>`"的详细说明，本节重点提示融资TS中对赌条款的核心博弈点：

::

   TS中对赌条款的核心要素：

   ① 业绩指标对赌
      - 营收/利润/GMV等具体数据里程碑
      - 若未达标：创始人补偿股份或现金

   ② 上市时间对赌
      - 约定X年内完成上市
      - 若未达标：投资人有权要求回购

   ③ 估值调整条款
      - 下轮融资估值不低于当前估值的一定倍数
      - 若未达标：现有投资人转换价格调整

   ④ 创始人激励对赌
      - 超越目标时投资人向创始人奖励股份
      - 双向对赌更为公平

----------------------------------

创始人最应关注的条款排名
========================

.. list-table:: 创始人条款关注优先级
   :header-rows: 1
   :widths: 10 25 30 35

   * - 排名
     - 条款
     - 原因
     - 建议策略
   * - ⭐⭐
     - **清算优先权**
     - 直接决定退出时的收益分配
     - 坚持1x非参与分配
   * - ⭐⭐
     - **董事会控制权**
     - 决定公司日常经营决策权
     - 争取创始人席位>投资人席位
   * - ⭐⭐
     - **防稀释条款**
     - Down Round时的保护机制
     - 接受加权平均，拒绝完全棘轮
   * - ⭐
     - **保护性条款**
     - 影响公司重大事项的决策效率
     - 设置金额门槛，避免过多否决项
   * - ⭐
     - **对赌条款**
     - 未达标对创始人个人风险极大
     - 争取双向对赌，设置合理目标
   * - ⭐
     - **股份锁定/转让限制**
     - 影响创始人流动性
     - 争取分期解锁，预留合理转让额度
   * - ▲
     - **优先购买权/共同出售权**
     - 影响股权转让自由度
     - 标准条款，关注细节即可
   * - ▲
     - **信息权/检查权**
     - 影响投资人信息披露负担
     - 约定合理的信息报送范围和频率

   ⭐⭐ = 必须全力争取的条款  ⭐ = 应予高度重视的条款  ▲ = 标准条款

实战案例：一份完整的A轮Term Sheet
====================================

以下是一份虚构但高度真实的A轮Term Sheet，展示各条款之间的互动关系及其对创始人利益的影响。

.. admonition:: 案例：云帆智能A轮融资条款清单
   :class: case-study

   **交易概览**

   ::

      目标公司：      云帆智能科技有限公司
     投资人：        星辰资本（领投方）
     跟投方：        远见创投、星河资本
     融资金额：      人民币6,000万元
     投前估值：      人民币4亿元
     投后估值：      人民币4.6亿元
     投资人持股比例：约13.04%

   **核心条款速览**

   .. list-table:: 核心条款一览
      :header-rows: 1
      :widths: 15 20 20 20 25

      * - 条款类别
        - 投资人要求
        - 创始人期望
        - 最终方案
        - 对创始人的影响
      * - 清算优先权
        - 1x参与分配
        - 1x非参与分配
        - **1x非参与分配**
        - 友好——退出时创始人收益较大
      * - 防稀释条款
        - 完全棘轮
        - 宽基加权平均
        - **宽基加权平均**
        - 友好——稀释效应最小
      * - 董事会构成
        - 2席创始人+2席投资人+1席CEO
        - 3席创始人+1席投资人
        - **2席创始人+1席投资人+1席CEO+1席独立**
        - 制衡——创始人在董事会占微弱多数
      * - 保护性条款
        - 12项否决权
        - 6项否决权
        - **8项否决权（设金额门槛）**
        - 可接受——重大事项需投资人同意
      * - 期权池
        - 预留20%（创始人承担）
        - 预留10%
        - **预留15%（创始人承担）**
        - 创始人隐性稀释约8.3%
      * - 对赌条款
        - 营收对赌
        - 无对赌
        - **双向对赌（营收+新客）**
        - 合理——达标有奖励，未达标适度补偿
      * - 排他期
        - 90天
        - 30天
        - **60天**
        - 标准期限
      * - 投资人类别
        - 领投方特殊权利
        - 各投资人权利一致
        - **各投资人权利基本一致**
        - 避免领投方过度特权

   **条款互动分析**

   以下可视化展示各条款如何共同影响创始人的实际利益：

   ::

      名义估值 vs 有效估值对比（对创始人）

      投前估值：                                   4.00亿
      减：15%期权池（由创始人承担）                 0.52亿
      ───────────────────────────────────────────────
      创始人实际享有价值的估值基础：                 3.48亿
                                                      ↓
      相当于名义估值打了约87折


      退出情景模拟（假设公司以10亿估值被收购）：

                      ┌─────────────────────────────────┐
                      │          10亿总对价              │
                      ├─────────────────────────────────┤
                      │  优先权分配（投资人收回6,000万）  │
                      │        剩余9.4亿                │
                      ├─────────────────────────────────┤
                      │  按持股比例分配:                │
                      │  ① 投资人（共16.6%）= 1.56亿     │
                      │     （包括优先权部分的6,000万）   │
                      │  ② 创始人团队（58.4%）= 5.49亿   │
                      │  ③ 期权池（15%）= 1.41亿         │
                      │  ④ 员工持股（10%）= 9,400万      │
                      └─────────────────────────────────┘

      若采用"1x参与分配"方案：
      投资人先收回6,000万，再分得1.56亿 = 总计2.16亿
      创始人仅分得约4.93亿 —— 差额约5,600万


      Down Round情景模拟（假设下轮估值降为2亿）：

      无防稀释条款：    创始人股权被大幅稀释
      宽基加权平均：    投资人转换价格调整约11%
                        创始人额外稀释约3-5%
      完全棘轮：        投资人转换价降至新股价
                        创始人可能被稀释15-25%

      → 本例采用宽基加权平均，创始人风险可控

   **谈判筹码分析**

   ::

      本轮TS谈判中创始人的关键筹码：

      ① 已有3家投资机构出具TS（竞争性谈判）
         → 成功争取到非参与分配+宽基加权平均

      ② 公司ARR已突破3,000万，增长约150%
         → 抵消了投资人关于"完全棘轮"和"参与分配"的要求

      ③ 有2家老股东愿意跟投
         → 降低领投方风险，换取更友好条款

TS谈判十诫
==========

::

   1. 估值不是唯一——条款比估值更重要
   2. 永远不要接受完全棘轮防稀释
   3. 清算优先权决定了公司"卖了之后"谁来拿钱
   4. 董事席位是权力的核心战场
   5. 对赌条款要设上限——创始人不应无限兜底
   6. 期权池的"存量"是创始人的隐性成本
   7. 排他期不宜过长（通常45-60天）
   8. 不要以为TS的不具约束力可以随便签
   9. 找有经验的律师审阅——这是值得花的钱
   10. 好的投资人懂你的业务，不只是懂条款
