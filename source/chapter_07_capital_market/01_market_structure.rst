-------------------------------

资本市场金字塔结构
-------------------------------

中国资本市场呈现典型的**金字塔型多层次结构**，从上至下依次为：

::

                   主板（上交所/深交所）
               ↑ 大型成熟企业，利润要求最高
           ──────────────────────
                  科创板（上交所）
               ↑ 硬科技公司，允许未盈利
           ──────────────────────
                  创业板（深交所）
               ↑ 成长型创业公司
           ──────────────────────
                   北交所
               ↑ "专精特新"中小企业
           ──────────────────────
              新三板（全国股转系统）
               ↑ 中小微企业，门槛最低

-------------------------------

各板块定位说明
-------------------------------

主板
--------

- **交易所**：上海证券交易所 / 深圳证券交易所
- **定位**：大盘蓝筹企业
- **服务对象**：大型成熟企业，行业龙头
- **特点**：利润要求最高，上市审核标准最为严格

科创板
--------

- **交易所**：上海证券交易所
- **定位**：硬科技企业
- **服务对象**：符合国家战略、突破关键核心技术、市场认可度高的科技创新企业
- **特点**：允许未盈利企业上市，强调科技属性

创业板
--------

- **交易所**：深圳证券交易所
- **定位**：成长型创新创业企业
- **服务对象**：高成长性的中小型创业公司
- **特点**：支持"三创四新"（创新、创造、创意；新技术、新产业、新业态、新模式）

北交所
--------

- **交易所**：北京证券交易所
- **定位**："专精特新"中小企业
- **服务对象**：创新型中小企业，具备专业化、精细化、特色化、新颖化特征
- **特点**：为中小企业提供专属融资平台

新三板（全国中小企业股份转让系统）
------------------------------------

- **交易所**：全国中小企业股份转让系统（NEEQ）
- **定位**：中小微企业
- **服务对象**：各类中小微企业
- **特点**：挂牌门槛极低（无利润要求，甚至无营收要求），是资本市场的"预备板"

-------------------------------

金字塔结构分析
-------------------------------

中国资本市场金字塔结构的设计理念体现了**分层递进、错位发展**的思路：

从塔尖到塔基，企业的规模和成熟度递减，但企业数量递增。主板处于金字塔顶端，上市门槛最高、企业数量最少（约3,000家）；新三板处于金字塔底部，门槛最低、企业数量最多（约6,000家）。

这种结构的设计目的在于：

1. **分层服务**：不同发展阶段的企业可以找到适合自己的融资平台
2. **升降通道**：企业成长后可升入更高层级（如新三板→北交所→沪深交易所）
3. **风险分层**：不同板块设置不同的投资者适当性要求，实现风险匹配
4. **错位竞争**：各板块差异化定位，避免同质化竞争

.. note::

   北交所成立后，中国形成了"沪深交易所 + 北交所 + 新三板"的完整多层次资本市场体系，
   覆盖从初创期到成熟期的全生命周期企业融资需求。

为什么中国需要多层次资本市场？
========================================

这并非偶然，而是中国经济发展阶段和融资结构矛盾的必然结果。

**深层原因一：中小企业融资难的结构性矛盾**

::

   中国有超过 5,000 万家中小企业，贡献了 60% 以上的 GDP
   但银行体系天然偏好大企业——中小企业缺乏抵押物、信用记录不完整
   股权融资成为中小企业唯一的出路

   但问题是：
     主板门槛太高 → 99% 的企业达不到
     如果没有低门槛板块 → 大量中小企业永远无法获得股权融资
     新三板/北交所的存在 → 给这些企业一个"进场"的机会

   所以，多层次资本市场的本质是：**用不同门槛覆盖不同发展阶段的企业**

**深层原因二：从"间接融资"到"直接融资"的国家战略**

::

   中国长期依赖银行体系（间接融资），企业融资主要靠贷款
   但这种模式有两个致命问题：
     1. 银行风险集中——房地产泡沫、地方债务风险都集中在银行体系
     2. 创新企业融不到钱——银行看不懂科技公司的商业模式

   多层次资本市场的建立，本质上是国家推动"去杠杆"和"科技创新"：
     让企业从"找银行借钱"转向"找市场融资"
     把风险从银行体系分散到资本市场

**深层原因三：各板块的"错位竞争"逻辑**

::

   为什么不是"一个统一的交易所"而是"多板块"？
   因为不同企业的融资需求差异太大了：

     大型国企：   融资规模大（百亿级），风险低 → 主板
     硬科技公司： 尚未盈利，但技术壁垒高 → 科创板
     成长型公司： 有利润，但规模小 → 创业板
     专精特新：   小而美，细分领域龙头 → 北交所
     小微企业：   尚未达标，先挂牌培育 → 新三板

   如果只有一个板块，要么门槛太高（小企业进不来），
   要么门槛太低（大企业不屑来），无法满足多样化需求。

对创始人的决策启示
========================

::

   问题：我的公司应该上哪个板块？
   答案：取决于"你离上市还有多远"

   阶段1：公司尚未盈利，营收<1亿
     → 新三板挂牌（先进入资本市场体系，哪怕只是"挂个名"）
     → 意义：规范公司治理、建立品牌信用、为后续转板打基础

   阶段2：公司已盈利，利润2,000-5,000万
     → 北交所（门槛最低的"真正上市"板块）
     → 或者继续在新三板创新层培育，等待转板

   阶段3：公司利润5,000万以上，营收3亿+
     → 创业板或科创板（取决于行业属性）
     → 硬科技→科创板，其他→创业板

   阶段4：公司利润1亿+，行业龙头
     → 主板（传统行业）或科创板（硬科技）

   关键洞察：**不是"哪个板块最好"，而是"哪个板块最适合你现在的状态"**
   选择上市板块的核心逻辑 = 先上车，再换座
   新三板→北交所→沪深，是递进式上市的典型路径

.. list-table:: 各板块对创始人的核心意义
   :header-rows: 1
   :widths: 15 30 25 30

   * - 板块
     - 对创始人的核心价值
     - 融资能力
     - 流动性
   * - **主板**
     - 品牌背书，融资规模最大
     - 极高（百亿级）
     - 极高
   * - **科创板**
     - 科技属性认证，估值溢价
     - 高（十亿级）
     - 高
   * - **创业板**
     - 成长性企业的主流选择
     - 高（十亿级）
     - 高
   * - **北交所**
     - 中小企业进入资本市场的"第一站"
     - 中（亿级）
     - 中
   * - **新三板**
     - 规范治理、品牌建设、为转板做准备
     - 低（千万级）
     - 极低

.. warning::

   新三板不能简单理解为"低配版上市"。它的流动性极差、融资功能弱，
   很多公司挂牌后发现"花了钱、没融到资、还增加了合规成本"。
   创始人决定是否上新三板时，需要权衡：**挂牌的规范收益 > 合规成本**？
